# Children's Book For All website

## Site Map and Website Features

### Sign Up

Users can sign up on the website with their name, email address and mobile number to stay updated on CBA activities.
Optionally, users can also update about themselves, how they got to know about CBA, the place they are joining from and any message they have for CBA.

### Register for a Reading

Users can see all the information about the reading and register for the book reading.

- User sees information about a book reading.
- User fills a form to register for the book reading - enter name, email, phone and amount they want to contribute for the book reading.
- User pays for the book reading using UPI
- User adds the reading on their calendar

### CBA Members

Users can see a paginated list of all the CBA members. User can click on any member to see their journey with Children's books.

### Past Readings

Users can see a paginated list of all the past CBA readings. Users can click on any reading to see the documentation for that reading.

### Gift a Children's Book

Users can choose to gift a book to themselves, their loved ones or CBA.
Users can fill a form to write theme, budget, receiver and sender details, and pay using UPI method.
